python -m http.server 8000 (to start the front end - important to use with cookies)


If you want to export the conda env so that you can have the same env on a different machine:
conda env export -n <env_name> -f environment.yml --no-builds

flask run --port=5001

python 1_migrate_users_to_db.py

python get_data_from_render.py
